## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open your browser and navigate to `http://localhost:4200`

### Build for Production

```bash
npm run build
```

The build artifacts will be stored in the `dist/` directory.

## Customization

### Update Personal Information

1. **Home Component** (`src/app/components/home/home.component.html`):
   - Update name, title, and description

2. **About Component** (`src/app/components/about/about.component.ts`):
   - Modify about points and statistics

3. **Skills Component** (`src/app/components/skills/skills.component.ts`):
   - Update skill categories and proficiency levels

4. **Projects Component** (`src/app/components/projects/projects.component.ts`):
   - Add your projects with details

5. **Experience Component** (`src/app/components/experience/experience.component.ts`):
   - Update work experience entries

6. **Contact Component** (`src/app/components/contact/contact.component.ts`):
   - Update contact information

### Styling

The main styles are in `src/styles.scss`. You can customize:
- Color scheme (CSS variables in `:root`)
- Typography
- Spacing and layout
- Animations
